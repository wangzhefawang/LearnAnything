# Agent、RAG 与知识系统

## Prompt 与上下文工程

- Prompt 工程
  - 角色设定、Few-shot、思维链 CoT、结构化输出
  - 设计科研问答中的系统提示词、任务提示词与约束提示词
- 上下文设计与管理
  - 系统提示词、对话历史裁剪、上下文压缩
  - 控制引用材料、用户问题与工具结果在上下文中的组织顺序
  - 处理长上下文截断、冗余信息与无关证据干扰

## RAG 与向量检索

- 向量化 Embedding
  - 将论文标题、摘要、段落或问题映射为稠密向量
  - 理解向量维度、归一化与批量编码的工程影响
  - 评估 embedding 模型对同义表达和跨句语义的捕捉能力
- 语义相似度计算
  - 掌握余弦相似度、点积与欧氏距离的差异
  - 理解向量归一化对相似度排序结果的影响
  - 分析语义匹配误召回与漏召回的常见原因
- 论文文档分块 Chunking
  - 按章节、段落、滑动窗口或语义边界切分论文长文本
  - 设置 chunk size、overlap 与元数据字段
  - 处理公式、表格、引用和跨段落依赖对分块质量的影响
- RAG 召回与重排流程
  - 构建 query embedding、向量召回、候选片段过滤的流水线
  - 使用 cross-encoder 或规则特征进行 reranking
  - 理解多路召回、去重和证据聚合对答案质量的作用
- 上下文注入与答案生成
  - 将检索片段、来源信息和回答约束注入 LLM 上下文
  - 要求模型基于证据回答并避免无依据扩展
  - 处理上下文过长、证据冲突和引用定位问题
- RAG 效果评估
  - 区分检索指标、生成指标和端到端问答准确率
  - 设计人工标注、基准问题集与消融实验
  - 评估答案相关性、忠实性、引用准确性和可复现性
- 文档解析与分块
- 向量数据库与检索
  - Chroma/Milvus/FAISS、相似度检索、混合检索

## 向量检索与 FAISS

- FAISS 向量索引构建
  - 理解向量入库、ID 映射、元数据回查与索引持久化
  - 掌握批量构建索引与增量更新的基本流程
  - 区分 CPU 索引与 GPU 加速检索的使用场景
- Flat、IVF、HNSW、PQ 索引类型
  - 理解 Flat 精确检索与近似检索的取舍
  - 掌握 IVF 聚类倒排、HNSW 图检索与 PQ 量化压缩的核心思想
  - 根据语料规模、延迟目标和内存限制选择索引类型
- 向量相似度度量选择
- 召回率与检索性能优化
  - 调节 top-k、nprobe、efSearch 等参数平衡召回与延迟
  - 设计基准集评估召回率、检索耗时和内存占用
  - 分析分块粒度、向量质量和索引参数对召回效果的影响

## 知识图谱与 GraphRAG

- 图数据模型与知识表示
  - 用节点、关系、属性表达论文、作者、方法、任务和指标
  - 设计 schema 约束与实体类型层次
  - 理解图结构查询相比纯向量检索的优势与边界
- Neo4j 与 Cypher 查询
  - 掌握 MATCH、WHERE、RETURN、MERGE 等基础语句
  - 编写多跳关系查询和路径查询
  - 理解索引、约束和查询计划对图查询性能的影响
- 科研知识图谱构建
  - 从论文语料中抽取实体、关系、属性和出处
  - 进行实体规范化、去重、消歧与关系校验
  - 建立从原文证据到图谱事实的可追溯链路
- GraphRAG 与 KG+LLM 融合
  - 结合图查询、向量检索和 LLM 生成回答复杂问题
  - 用知识图谱约束答案实体、关系和推理路径
  - 设计图谱证据与文本证据的融合策略

## Agent 智能体

- Function Calling 工具调用
  - 定义检索、图查询、摘要、重排等工具的输入输出 schema
  - 处理参数生成、类型校验、异常返回和重试
  - 控制工具结果如何回填给 LLM 继续推理
- ReAct 推理与行动循环
  - 理解 Thought、Action、Observation 的基本循环
  - 让 Agent 根据中间观察决定继续检索、查询图谱或生成答案
  - 识别循环失控、工具误用和无效推理链的问题
- 长文本问答策略
  - 使用分层检索、逐段摘要和证据合并处理论文长文本
  - 针对综述型、事实型、比较型问题选择不同回答流程
  - 控制答案结构、引用来源和不确定性表达
- Agent 架构基础
- 任务拆解与规划
- 记忆机制
  - 短期对话记忆、长期持久化记忆
- 多智能体协同
